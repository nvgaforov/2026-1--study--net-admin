# Цель работы

## Цель

Освоить настройку прав доступа пользователей  
к ресурсам сети с помощью расширенных ACL  
в Cisco Packet Tracer.

## Настройка ACL для серверной сети

- На маршрутизаторе создан ACL **servers-out**
- Разрешён доступ:
  - ко **web-серверу** по **HTTP**
  - к **mail-серверу** по **SMTP** и **POP3**
  - к **DNS-серверу** по **UDP/53**
  - в сторону серверов по **ICMP**
- Администратору дополнительно открыт доступ
  по **FTP** и **Telnet**

![Настройка ACL servers-out](Screenshot_4.png){ width=84% }

## ACL для сети Other и сети управления

- Создан ACL **other-in**
  - пользователям сети **Other** запрещён выход
    за пределы своей сети
  - для администратора оставлено разрешение
- Создан ACL **management-out**
  - доступ к сети **10.128.1.0/24**
    разрешён только администраторам

![Настройка ACL other-in и management-out](Screenshot_5.png){ width=84% }
