Цель работы

Цель

Настроить статическую маршрутизацию VLAN в сети
и изучить передачу пакетов между виртуальными сетями
в Cisco Packet Tracer.

Ход выполнения

Топология сети

  • В проект добавлен маршрутизатор Cisco 2811
  • Маршрутизатор подключён к порту Fa0/24 коммутатора msk-donskaya-ngaforov-sw-1

Базовая настройка маршрутизатора

Настройка Router-on-a-Stick

  • На интерфейсе FastEthernet0/0 созданы подинтерфейсы
  • Для каждого подинтерфейса настроена инкапсуляция dot1Q
  • Подинтерфейсы соответствуют VLAN 101–104
  • Каждому VLAN назначен IP-адрес шлюза

Проверка связности между VLAN

  • Выполнены ping-запросы к узлам из других VLAN
  • Узлы успешно отвечают на ICMP-запросы
  • Первый пакет теряется из-за ARP-разрешения
  • Межвлановая маршрутизация работает корректно

Режим Simulation

  • Создан тестовый обмен ICMP между узлами
  • Передача пакета отслежена пошагово
  • Пакет проходит через коммутаторы и маршрутизатор
  • Наблюдается переход трафика между VLAN

Анализ структуры пакета

  • Изучено окно PDU Information
  • Передача осуществляется в кадре Ethernet 802.1Q
  • Кадр содержит VLAN-тег
  • Внутри кадра передаётся пакет IPv4 и сообщение ICMP
