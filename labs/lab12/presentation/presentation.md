# Цель работы

## Цель

Приобретение практических навыков  
по настройке NAT и ограничению доступа  
в локальной сети.

# Ход выполнения

## Настройка маршрутизатора провайдера

- Включены интерфейсы (**no shutdown**)  
- Настроен подинтерфейс:
  - **Fa0/0.4**
  - VLAN 4 (dot1Q)
- IP:
  - 198.51.100.1 /28  
- Внешний интерфейс:
  - 192.0.2.1 /24  

![Настройка provider-gw-1](Screenshot_1.png){ width=70% }

## Настройка коммутатора

- Порты переведены в режим **trunk**
- Создана VLAN 4 (**nat**)
- Настроен интерфейс **Vlan4**

![Настройка provider-sw-1](Screenshot_2.png){ width=70% }

## Подключение сети «Донская»

- Настроен интерфейс:
  - **Fa0/1.4**
- IP:
  - 198.51.100.2 /28  
- Добавлен маршрут по умолчанию:
  - 0.0.0.0 → 198.51.100.1  

![Подключение к провайдеру](Screenshot_3.png){ width=70% }

## Настройка NAT

- Создан пул:
  - 198.51.100.2–198.51.100.14  
- Настроен ACL  
- Включён PAT (overload)

![Настройка NAT](Screenshot_4.png){ width=70% }

## Статический NAT

Настроен проброс портов:

- HTTP → 10.128.0.2  
- FTP → 10.128.0.3  
- SMTP/POP3 → 10.128.0.4  
- RDP → 10.128.6.200  

![Статический NAT](Screenshot_5.png){ width=70% }
